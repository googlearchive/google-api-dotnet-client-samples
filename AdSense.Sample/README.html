1. Checkout Instructions

Prerequisites: Install Visual Studio, and Mercurial.

cd [someDirectory] 
hg clone https://code.google.com/p/google-api-dotnet-client.samples/ google-api-dotnet-client-samples

2. API access Instructions

Important: after checking out the project, and before compiling and running it, you need to:

  • Create a project in Google APIs console
  • Activate the AdSense Management API for your project

Set up Project in Visual Studio

  • Open the GoogleApisSamples.sln with Visual Studio
  • Click on Build > Rebuild Solution
  • Execute the .exe in AdSense.Sample\bin\Debug
  • When prompted insert your client credentials which are generated on the Google APIs Console API Access pane.

Documentation

If you want to learn more about the API visit the AdSense Management API documentation.
